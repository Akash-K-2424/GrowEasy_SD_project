# GrowEasy CSV Importer -- Frontend

Next.js (App Router) + TypeScript + Tailwind UI for the AI-powered CSV importer.
See the [repo root README](../README.md) for the full project overview, setup
instructions, and API docs.

```bash
npm install
cp .env.local.example .env.local   # set NEXT_PUBLIC_API_BASE_URL
npm run dev
```
