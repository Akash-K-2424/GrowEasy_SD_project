# GrowEasy CSV Importer -- Backend

Express + TypeScript API: CSV parsing, batched AI extraction, CRM schema
validation. See the [repo root README](../README.md) for the full project
overview, setup instructions, and API docs.

```bash
npm install
cp .env.example .env   # set LLM_PROVIDER + matching API key
npm run dev            # http://localhost:4000
npm test                # 49 unit tests
```
